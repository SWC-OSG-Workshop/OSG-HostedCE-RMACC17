Overview

Requirements

Integrating your cluster into OSG has fairly low requirements. Your cluster should meet the following:

  • Slurm, PBS, SGE, LSF, or HTCondor job scheduler
  • Submit node should be accessible using SSH
  • Compute nodes should have internet connectivity (NAT is fine)
  • Compute nodes should use RHEL/Centos/Scientific Linux 6 or 7

Cluster Information

Please fill out this questionnaire so that we have information about your cluster before the workshop. This will give us extra time to configure the OSG managed services for your cluster. Additionally, you should create the user that the OSG infrastructure will use to submit jobs to your cluster in order to facilitate the integration process during the workshop.
